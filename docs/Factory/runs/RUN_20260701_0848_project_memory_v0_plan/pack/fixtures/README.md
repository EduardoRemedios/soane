# Golden Fixture Suite

## Version

v1

## Change Log

- v1 (2026-07-01): Initial Stage F fixture suite index.

## Purpose

These fixture definitions are planning assets for the future Project Memory v0 prototype.

They describe the minimum behavior that implementation fixtures and tests must prove. They are not executable tests yet.

## Fixtures

| Fixture ID | Required Behavior |
| --- | --- |
| `GF-001` | Decision linked to Evidence |
| `GF-002` | Assumption superseded by Evidence |
| `GF-003` | Contradiction between sources remains explicit |
| `GF-004` | Stale Evidence does not support current truth silently |
