# Raw Brief: Vision And Epistemic Model Hardening

## Request

Review and strengthen Soane's constitutional vision and canonical architecture after a repository-grounded critique identified gaps in epistemic semantics, delegated judgement, decision learning, memory governance, cross-project knowledge, and product success measures.

## Approved Direction

The human agreed to proceed with the recommendations from the 2026-07-12 vision review.

The bounded slice should:

1. Reframe Project Memory as the governed system of record for the Workspace's current understanding of a Project, without claiming authority over external reality or source-system records.
2. Define Claim or Knowledge Statement semantics so facts are reviewed epistemic status rather than an unsupported object label.
3. Replace the binary "humans think, AI works" framing with explicit human accountability plus bounded, revocable agent judgement under authority, policy, evidence, budget, and escalation conditions.
4. Define a decision outcome-review loop that compares expected consequences with observed outcomes and feeds operational learning.
5. Give memory rights, knowledge scope, and cross-project promotion constitutional treatment.
6. Strengthen Decision framing, canonical Markdown authority modes, product success measures, enterprise-scaling language, and voice/accessibility wording.
